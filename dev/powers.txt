Statuts :

id : power_exhaustion_status
nom (fr, en) : épuisement, exhaustion
Cibles possibles : Pouvoirs
Description mécanique : Statut simple qui annule les effets du pouvoir affecté. Compteur = tours restants avant suppression, décrémente de 1 à chaque fin de tour. Ce n'est pas ce statut qui est utilisé après activation d'un pouvoir lors de la résolution du plateau.
description en jeu : Empêche l'activation de tout effet.

id : hero_poison_status
nom (fr, en) : poison, poison
Cibles possibles : Héros (ennemi, duo)
Description mécanique : En fin de tour, inflige un nombre de dégâts imbloquables égal au compteur au héros affecté, puis décrémente de 1.
description en jeu : À la fin du tour, inflige {compteur} dégâts imbloquables puis réduit de 1."
